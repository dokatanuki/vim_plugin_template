# Vim Plugin Template

## Overview
Python3を使ったNeoVim Pluginのテンプレート  

## Directory
### `plugin/`
起動時に呼ばれるスクリプト  
### `autoload/`
必要に応じて読み込まれるスクリプト  
### `rplugin/python3/`
python3スクリプト
